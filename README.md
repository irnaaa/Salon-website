# Beauty Loft Salon Website

## Description
Beauty Loft is a modern and responsive salon website designed to showcase services, staff, and provide an easy way for clients to contact or book appointments.  
The website is optimized for desktop, tablet, and mobile devices, offering a seamless browsing experience.

## Features
- **Home / Hero Section:** Welcoming tagline and call-to-action button to book appointments.
- **Servicii (Services):** Displayed in interactive cards with "Vezi mai mult" buttons for detailed information.
- **Despre Noi (About Us):** Salon story, team introduction, and salon image.
- **Contact:** Contact form for inquiries and suggestions.
- **Modal Popups:** Detailed information about services and staff members.
- **Responsive Design:** Works perfectly on all screen sizes.
- **Interactive Elements:** Smooth animations and hover effects.

## Technologies Used
- **HTML5** – Website structure
- **CSS3** – Styling, layout, and animations
- **JavaScript** – Interactive cards, modal popups, and form functionality
- **Images & Media** – Salon photos and service images
